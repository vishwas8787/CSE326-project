I am Vishwas Saini, a first year student of B.Tech CSE. I have made a typing speed test website as my project of CSE326 subject using HTML, CSS and JavaScript.

This website has multiple difficulties (beginner, intermediate, advanced) so you can select whichever difficulty you feel comfortable with. After selecting the difficulty you can start the test.

There is a 60s timer and you have to complete test within this time. If you do not complete test within 60s, the test will end automatically. If you complete the test within 60s then the test ends automatically and result will be shown.

Once the test is done you will see your words per minute (WPM) and accuracy.
